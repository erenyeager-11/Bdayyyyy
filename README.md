A feature-loaded static birthday website made for anyone.

## Use

1. Put your audio file in this folder and name it `song1.mp3`.
   You can add more songs as `song2.mp3`, `song3.mp3`, up to `song8.mp3`; the player queue will show the files that actually exist.
2. Open `index.html` in a browser.
3. Send the whole folder or host it somewhere static so `index.html`, `styles.css`, `script.js`, `assets`, and your song files stay together.

## Files

- `index.html` - page structure and birthday sections
- `styles.css` - responsive layout, animated icons, cake, player, gifts, and visual polish
- `script.js` - music player, confetti, fireworks, wishes, guestbook, photo wall, countdown, and mini game
- `assets/birthday-stage.png` - birthday stage artwork
- `assets/birthday-album-cover.png` - local album cover artwork
